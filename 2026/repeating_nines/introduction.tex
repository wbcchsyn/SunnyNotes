\begin{frame}[c]{}{}
  What do you think the result of the following calculation is?

  \vspace{2ex}

  $1 \div 3 \times 3$

  \vspace{2ex}

  You would probably say 1.

  \vspace{2ex}

  What if someone told you,

  ``The answer is not 1''?
\end{frame}

\begin{frame}[c]{}{}
  {\large Proof}
\end{frame}

\begin{frame}[c]{}{}
  $1 \div 3 = 0.3333...$

  $0.3333... \times 3 = 0.9999...$

  \vspace{2ex}

  $\therefore 1 \div 3 \times 3 = 0.9999...$
\end{frame}

\begin{frame}[c]{}{}
  Also,

  \vspace{2ex}

  $0.9 < 1$

  $0.99 < 1$

  $0.999 < 1$
  
  ...

  $0.9999... < 1$
\end{frame}

\begin{frame}[c]{}{}
  Therefore

  \vspace{2ex}

  $1 \div 3 \times 3 < 1$
\end{frame}

\begin{frame}[c]{}{}
  Of course, this proof is wrong.

  \vspace{4ex}

  That's because

  $1 \div 3 \times 3 = 1$
\end{frame}

\begin{frame}[c]{}{}
  But where exactly did the proof go wrong?

  \vspace{4ex}

  Today, we're going to debunk this proof.
\end{frame}
